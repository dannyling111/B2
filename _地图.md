## 📋 分析摘要

### 原站問題
| 問題 | 影響 |
|------|------|
| 6 個獨立子頁面 | 用戶需反覆跳頁，流失率高 |
| 導航欄在小屏截斷 | 常見問題等重要頁面不可見 |
| 無黏性導航 | 滾動後「立即預約」消失 |
| 咩料/咩聊/咩療無說明 | 用戶不明白品牌核心概念 |
| 輔導員無照片 | 缺乏信任感 |
| 沒有 FAQ 頁面集成 | 用戶需另開頁面查詢 |

### v2 設計亮點
- 暖米白背景（#F5F0EB）+ 金色點綴（#C9A970）— 對標 BetterHelp 質感
- 咩料/咩聊/咩療 三欄交替佈局，每欄佔滿視口高度，保留完整詩意原文
- 滾動入場動畫（Intersection Observer，純 CSS transition）
- FAQ 手風琴（含危機熱線格式化展示）
- WhatsApp 深鏈接（wa.me/85262201625）
